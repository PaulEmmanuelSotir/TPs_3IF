COLLECT'IF
========

<img src="http://i.imgur.com/yPqRj5g.png" alt="Collect'if Logo" width="250"/>

Description
--------

Vous souhaitez faire une partie de tarot jeudi prochain ? Un match de volley le samedi de la 
semaine prochaine ? COLLECT’IF, l'association évenementielle locale, vous trouve des 
co-équipiers et met à votre disposition un local pour votre rencontre. 

Architechture générale du projet
--------

* Le répertoire contient un dossier GUI_Prototype regroupant les maquettes d'interface graphique Web (desktop et mobile).
* Le fichier '_CollectifData.sql' contient des données d'exemple pour la base de donnée SQL
* Le dossier 'Projet' contient le projet Maven constitué du code JAVA des services et de l'IHM sur console
* Le dossier 'BDD' contient la base de données SQL utilisée pour la persistance des objets de l'application
* Le dossier 'Docs' Contient une documentation plus détaillée du projet
